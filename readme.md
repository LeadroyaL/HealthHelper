#### 工作流程

客户端

1. 用蓝牙MAC地址向服务器换Token，即临时的AES Key
2. 建立Socket连接，不断去读蓝牙buffer的数据
3. 解析数据，用protobuf包装，之后用AES Key加密，上传到服务器

服务器

1. 建立MAC-Token的对应关系
2. 收到密文后解密，用protobuf解封，存入数据库
3. 提供数据库查询功能
